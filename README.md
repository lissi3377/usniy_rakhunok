# Тренажер «Усний рахунок»

Адаптивна гра для тренування усного множення і ділення. Працює без сервера, бази даних і зовнішніх аудіофайлів.

## Можливості

- автоматична генерація прикладів на множення й ділення на 5, 15, 20, 25, 0,5, 0,25 та 0,2;
- коротка стратегія обчислення після помилки, завершення часу або натискання кнопки «Підказка»;
- 10 рівнів: після кожних 5 правильних відповідей час скорочується;
- бали та серія правильних відповідей;
- звуки правильної відповіді, помилки, нового рівня та останніх секунд;
- підтримка крапки й коми в десяткових відповідях;
- адаптація до телефона, планшета та комп’ютера.

## Як опублікувати на GitHub Pages

1. Створіть новий публічний репозиторій на GitHub.
2. Завантажте в корінь репозиторію файл `index.html`.
3. Відкрийте **Settings → Pages**.
4. У розділі **Build and deployment** виберіть **Deploy from a branch**.
5. Виберіть гілку **main**, папку **/(root)** і натисніть **Save**.
6. Через кілька хвилин посилання на гру з’явиться на цій самій сторінці.

Звук починає працювати після натискання «Почати гру», оскільки браузери блокують автоматичне відтворення до першої дії користувача.
